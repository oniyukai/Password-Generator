### 密碼產生器
程式作者：yukai
程式名稱：密碼產生器 Version 3.0_2022.06.20
簡易的產生亂數密碼。使用自學的python語言編寫。
可以設定長度、組數、要有哪些字符。

### 使用方法
執行密碼產生器。
根據提問設定產生的密碼。
注意前兩題只能輸入半形數字。
如果再輸入而外字符裡有重複附加的字符，該字符產生的機率會比其他字符高。

關閉程式直接關閉即可。

切記要保管好密碼，不要外洩。小心詐騙。

### 日誌
- Version 3.0_2022.06.20

使用python的Tkinter，做出GUI介面化程式。開發和測試時沒有發現BUG與正常使用的限制，密碼長度和組數只能輸入半形數字。

停止開發這項計畫。
- Version 2.2_2022.06.19

改使用使用python語言編寫。

修正"輸入而外字符"的回答中，使用全行字符造成的錯誤。當出現回答內容錯誤時，會直接關閉程式。取消每次開啟程式生成的次數限制。
- Version 2.1_2022.05.14

修正了採用字元中最後一個無法被採用生成問題，也就是變數alphanum的最後一個字元不會採用的問題。
- Version 2.1_2022.04.02

修正了在是非題回答裡輸入二個字元以上(含)所出現的跳題Bug。增加每次開啟程式生成的次數限制，解決前兩題輸入非數字時的無窮迴圈，但能未解決前兩題輸入非數字字符所導致的錯誤。

2022.05.14發現在變數alphanum的最後一個字元不會採用。
- Version 2.0_2022.04.02

改使用使用C++語言編寫，檔案格式為.exe，大幅度降低電腦耗能，沒有了最多生成組數限制，可以選擇要加入哪些字符(1-9 ; a-z ; A-Z ; !@#$%^&*)，和填入自己而外想要加入的字符。

"輸入而外字符"的回答中，使用全形字符會造成錯誤。
- Version 1.0_2022.04.01

使用Microsoft Excel的巨集功能與Visual Basic for Applications (VBA)做出的小程式，檔案格式為.xlsm，使用Excel開啟，比較消耗電腦效能，一次最多能產生8組亂數密碼，有四種模式(數字 ; 數字&小寫字母 ; 數字&大小寫字母 ; 數字&大小寫字母&"@#$%^&*")。
